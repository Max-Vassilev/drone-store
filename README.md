# travel-agency

Frontend: React (Vite)
Backend: Flask
Database: PostgreSQL
Deployment: AWS

Currently, the website is deployed using an S3 bucket, EC2 instance, and RDS database.
The next step involves integrating an ASG, ALB, and CloudFront for the S3 bucket.
